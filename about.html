Public data only

Live Map shows source-linked public events, not certainty.

Data may be delayed, incomplete, unavailable, duplicated, or corrected by the original publisher. Verify important details with the original source.

Use Limits

Live Map is public-information awareness software. It is not for emergency response, navigation, financial decisions, legal decisions, medical decisions, or professional risk advice.

Source Status

Provider status can be live, degraded, fallback, disabled, or unavailable. A partial feed can still show useful events while some public providers are down.

Source Links

Event cards should keep source attribution and original links visible. When a provider is unavailable, the app should say so instead of pretending coverage is complete.

Free Data Rule

The default project uses APIs, feeds, downloads, RSS/Atom, GeoJSON, CSV, or official bulk datasets. Paid providers, secrets, and licensed scraping are not part of the core demo.

Status Meanings

Live
The provider returned usable public data recently.
Degraded
The provider responded only partially, slowly, or with stale data.
Fallback
The app is using a simpler backup path so the map remains useful.
Disabled
The provider is intentionally off, usually because it is planned, gated, or not safe for default use.
Unavailable
The provider failed or returned no usable data for this refresh.
