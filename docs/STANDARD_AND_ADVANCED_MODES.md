# Standard And Advanced Modes

Standard mode is the default for new visitors. It shows map, search, presets, essential filters, event feed, country selection, time window, and Live Tracking status.

Advanced mode exposes the existing analyst controls, including source health details, saved views, detailed grouping and sorting, provider/source diagnostics links, and technical record-kind language.

The selected mode is stored locally in browser storage. It is not an account setting.
