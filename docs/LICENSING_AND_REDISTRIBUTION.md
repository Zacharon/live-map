# Licensing and Redistribution

Live Map must not assume that public web access means automated collection, caching, commercial use, or redistribution is allowed.

## Default Rule

If commercial use or redistribution is unknown, keep the source planned, disabled, link-only, authentication-required, or license-required. Do not label it live.

## Before Ingestion

Confirm:

- Access rights are documented.
- Attribution text is implemented.
- Commercial-use status is known.
- Redistribution status is known.
- Cache and retention limits are recorded.
- Provider errors are sanitized.
- Credentials remain server-side.
