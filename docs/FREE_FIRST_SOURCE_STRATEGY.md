# Free-First Source Strategy

The baseline uses open feeds and public metadata before paid services: allowlisted RSS/Atom/JSON Feed, GDELT, Hacker News, Bluesky public AppView, and Wikimedia. YouTube, Mastodon, Twitch, and Kick remain server-side configuration boundaries because access, quotas, instances, or channel allowlists require explicit review.

No paid service was added in this phase. Provider schedules use small daily budgets, cache windows, and backoff; a feature flag alone does not override provider terms or legal review.
