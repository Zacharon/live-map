# Global Fishing Watch Provider

Global Fishing Watch is the preferred maritime activity boundary for open ocean-analysis features.

Environment variables:

```text
GFW_ENABLED=
GFW_API_TOKEN=
```

This phase includes normalization and provider-status boundaries, but live endpoint activation remains configuration-required until authorization, attribution, endpoint scope, delay, and redistribution requirements are documented.

Do not claim complete real-time global AIS coverage from Global Fishing Watch. Label ocean activity, vessel events, fishing activity, encounters, loitering, port visits, and vessel-presence analysis according to the endpoint and delay actually used.
