# Consumer UX

The public default is a global news map, not an analyst console.

Primary areas:

- Explore: clean map with important verified events.
- Live Feed: chronological news and incidents.
- Country Scores: country rankings, trends, and profiles.

Tools and research features remain available under Tools or Advanced mode: diagnostics, Source Explorer, provider status, saved views, raw observations, and detailed taxonomy controls.

Standard mode hides provider IDs, request budgets, adapter status, raw observations, discovery leads, and planned-provider details.
