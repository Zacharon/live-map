# Provider Implementation Roadmap

Recommended order after the registry foundation and the current live GDACS integration:

1. ReliefWeb humanitarian events
2. GDELT major-news discovery
3. CISA KEV and NVD technology/cyber events
4. SEC EDGAR finance events
5. EIA commodity and energy events
6. ACLED conflict events after credentials and licensing approval
7. Aircraft tracking after performance and terms review
8. Maritime tracking after account and redistribution review

Do not add many unrelated feeds before the interface can explain source status, freshness, licensing, verification, and confidence.
