# Maritime Reference Data

Reference and event boundaries for maritime context include:

- IMO GISIS.
- Equasis.
- USCG Navigation Center.
- NOAA nautical charts.
- World Port Index.
- UKMTO advisories.
- U.S. Maritime Administration advisories.
- Port authority alerts.
- Notices to Mariners.
- Coast-guard feeds.

Static port locations are reference data. They must not be merged with active vessel positions.
