# Social Platform Access Matrix

| Platform | Status | Boundary |
| --- | --- | --- |
| YouTube | Configuration-required | Official Data API, API key, approved watchlists, metadata only. |
| Bluesky | Configuration-required | Public AppView search, bounded watchlists, metadata only. |
| Mastodon | Configuration-required | Approved instance and watchlists, metadata only. |
| Hacker News | Configuration-required | Official/public story metadata, bounded queries. |
| Wikimedia | Configuration-required | Recent-change metadata only. |
| Twitch | Configuration-ready | Approved channel allowlist and official access required. |
| Kick | Configuration-ready | Approved access and channel allowlist required. |
| X, TikTok, Instagram, Facebook, Threads, Reddit, Rumble | Registry/link-only or disabled | No scraping, bypasses, automation, or fake live status. |
